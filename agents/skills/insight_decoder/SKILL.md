# Skill: Insight Decoder

## Description
This skill analyzes the Jobs To Be Done (Functional, Emotional, Social) and identifies the core Insight (Tension, Truth).

## Input (Context)
- `input_text`: The raw situation text.
- `contextGame`: Result from Context Game skill.

## Output (Context)
- `decoder`: { jobs, tension, insight }

## Template
- `default.txt`: Standard prompt for insight decoding.
