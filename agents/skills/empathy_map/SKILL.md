# Skill: Empathy Map

## Description
This skill steps into the customer's shoes to create an Empathy Map (See, Hear, Say & Do, Think & Feel).

## Input (Context)
- `input_text`: The raw situation text.

## Output (Context)
- `empathyMap`: { see, hear, say_and_do, think_and_feel, pain, gain }

## Template
- `default.txt`: Standard prompt for empathy mapping.
